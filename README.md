# Active Directory Home Lab

## Overview
This project demonstrates setting up a Windows Server 2022 domain controller using VirtualBox.

## Steps Completed

### 1. VM Setup
- Created VM in VirtualBox
- Allocated 4GB RAM, 2 CPUs, 40GB storage

### 2. Installed Windows Server 2022
- Used Desktop Experience version

### 3. Configured Static IP
- IP: 192.168.1.10
- DNS: 127.0.0.1

### 4. Installed Active Directory
- Promoted server to Domain Controller
- Created domain: mydomain.local
